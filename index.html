Angular 1 是我初次接觸的 JavaScript 框架。它標榜的 MVC 架構啟發我、並讓我提升到另一個境界。

但是，大家終究會拋棄 Angular 1。而 Google 力推的 Angular 2 也和 Angular 1 相差甚大。尤其，Angular 2 使用需要編譯的 TypeScript。

這使我考慮放棄 Angular 2、並轉向其他框架。

不久，我發現語法和 Angular 1 類似的 Vue，但它更為靈活而簡潔。如果想考慮其他框架的話，Vue 會是個好選擇。

總之，因為 Angular 1 已經要走向歷史，所以這個專案以後不會再維護了。

有緣再會。

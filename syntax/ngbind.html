ng-bind 就像是寫在 HTML 裡面的大括弧。但考慮到要是瀏覽器停用 JavaScript 的話，雙大括弧就會「原形畢露」，所以會有人用 ng-bind 載入控制器裡面的變數。如果碰上瀏覽器停用 JavaScript──這意味著 ng-bind 失效──瀏覽器就會載入原本寫在該元件的內容。就像 p ng-bind="crte" 那樣。

  • 把綁定的變數放在 ng-bind 裡面，效果和放在元件內的大括弧一樣。
  • 如果要測試效果，請使用 Web Developer 套件裡面的「停用 JavaScript」。

foobar in CSS style

foobar in ng-style

ng-style：可以運用 JSON 的形式，指派元件的 CSS 樣式。

  • 大括弧與 return 必須同一行，不可換行，不然辨識會出問題。
  • JavaScript 的減號 - 不是合法的變數名──那要怎麼套用如 background-color 或 text-align 的東西？答案是去除減號後，第二個單詞首字以大寫表示：例如 backgroundColor 或 textAlign。

以下的顯示效果是相同的：

.foobar
{
    margin:10px;
    background-color:#C87630;
}
<p class="foobar">foobar in CSS style</p>

$scope.foobar = function ()
{
    return {
        margin:"10px",
        backgroundColor:"#C87630",
    };
};
<p ng-style="foobar()">foobar in ng-style</p>
